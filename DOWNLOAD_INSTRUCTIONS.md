# Al-Idrisi Text Download Instructions

## Overview

This repository contains information about Al-Idrisi's 12th-century masterwork "Nuzhat al-Mushtaq fi Ikhtiraq al-Afaq" (also known as the Tabula Rogeriana or The Book of Roger). While the full text files could not be automatically downloaded due to network restrictions, complete public domain versions are freely available from multiple sources.

## Available Versions

### 1. French Translation (Complete, Public Domain)
**Translation by Pierre-Amédée Jaubert (1836-1840)**

This is the first complete translation of Al-Idrisi's work and remains one of the most accessible versions. Published in the 19th century, it is fully in the public domain.

- Publisher: ARC Humanities Press (May 2024)
- ISBN: 9781802700190

- Available through academic libraries

## Quick Start: How to Download

### Option 1: Internet Archive (Easiest)
1. Visit: https://archive.org/details/gographiededrisi01idri
2. Click "Download Options" on the right sidebar
3. Choose your preferred format:
   - **PDF** - Best for printing or offline reading
   - **EPUB** - Best for e-readers (Kindle, Kobo, etc.)
   - **Plain Text** - Best for text processing or analysis
   - **DjVu** - Compact image-based format

### Option 2: Wikimedia Commons (Direct PDF)
1. Visit: https://upload.wikimedia.org/wikipedia/commons/2/2b/Edrisi_-_Géographie,_traduite_par_P._Amédée_Jaubert,_tome_second.pdf
2. Your browser will display or download the PDF directly
3. Save using your browser's save function

### Option 3: Command Line Download

If downloading from a system with internet access, you can use:

```bash
# Download Volume 1 (French) - PDF
curl -L -o "al-idrisi-vol1-french.pdf" \
  "https://archive.org/download/gographiededrisi01idri/gographiededrisi01idri.pdf"

# Download Volume 2 (French) - PDF
curl -L -o "al-idrisi-vol2-french.pdf" \
  "https://archive.org/download/gographiededrisi02idri/gographiededrisi02idri.pdf"

# Download Volume 1 (French) - Plain Text
curl -L -o "al-idrisi-vol1-french.txt" \
  "https://archive.org/stream/gographiededrisi01idri/gographiededrisi01idri_djvu.txt"

# Download Volume 2 (French) - Plain Text
curl -L -o "al-idrisi-vol2-french.txt" \
  "https://archive.org/stream/gographiededrisi02idri/gographiededrisi02idri_djvu.txt"

# Download from Wikimedia Commons
curl -L -o "al-idrisi-vol2-wikimedia.pdf" \
  "https://upload.wikimedia.org/wikipedia/commons/2/2b/Edrisi_-_Géographie,_traduite_par_P._Amédée_Jaubert,_tome_second.pdf"
```

## Copyright Status

All versions of Jaubert's French translation (1836-1840) are in the **public domain** worldwide, as they were published over 180 years ago. The Arabic original (1154 CE) is also public domain.

The 2024 English translation is under copyright and must be purchased.

## Technical Note

During the initial compilation of this repository, automatic downloads were blocked due to network proxy restrictions (HTTP 403 errors). However, these files are freely accessible from standard web browsers and download tools on systems with normal internet access.
